Documentation

Cette API permet d'évaluer la performance énergétique des logements situés dans le département du Rhône. Elle est conçue pour obtenir rapidement une estimation du coût annuel de chauffage ainsi qu'une prédiction de l’étiquette DPE (Diagnostic de Performance Énergétique) d’un bien immobilier.

URL :

https://france-energie.koyeb.app/api/v1

Prédiction du cout du chauffage

GET

/cout_chauffage

Paramètres :

  • valeur_1
  • valeur_2
  • valeur_3

Prédiction de l'étiquette DPE

GET

/etiquette_dpe

Paramètres :

  • valeur_1
  • valeur_2
  • valeur_3

Utilisation

cURL

curl -G "https://france-energie.koyeb.app/api/v1/cout_chauffage" \
    --data-urlencode "valeur_1=30 rue de la republique, lyon" \
    --data-urlencode "valeur_2=55" \
    --data-urlencode "valeur_3=2024"

Python

import requests

base_url = "https://france-energie.koyeb.app/api/v1"
endpoint = "/cout_chauffage"

params = {
    'valeur_1': "30 rue de la republique, lyon",
    'valeur_2': 55,
    'valeur_3': 2024
}

response = requests.get(base_url + endpoint, params=params)

R

library(httr)

base_url <- "https://france-energie.koyeb.app/api/v1"
endpoint <- "/cout_chauffage"

response <- GET(
    url = paste0(base_url, endpoint),
    query = list(
    valeur_1 = "30 rue de la republique, lyon",
    valeur_2 = 55,
    valeur_3 = 2024
    )
)
