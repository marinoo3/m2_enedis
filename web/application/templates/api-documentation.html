Documentation

Cette API permet d'évaluer la performance énergétique des logements situés dans le département du Rhône. Elle est conçue pour obtenir rapidement une estimation de la consommation annuelle de chauffage ainsi qu'une prédiction des logements passoires (étiquette F-G).

URL :

https://france-energie.koyeb.app/api/v1

Prédiction de la consommation de chauffage (kwh)

GET

/consommation_chauffage

Paramètres :

  • surface_habitable_logement
  • altitude
  • annee_construction
  • zone_climatique
  • type_batiment
  • qualite_isolation_enveloppe
  • type_energie_principale_chauffage
  • logement_traversant
  • protection_solaire_exterieure

Prédiction de l'étiquette DPE

GET

/etiquette_dpe

Paramètres :

  • surface_habitable_logement
  • altitude
  • annee_construction
  • zone_climatique
  • type_batiment
  • qualite_isolation_enveloppe
  • type_energie_principale_chauffage
  • logement_traversant
  • protection_solaire_exterieure

Utilisation

cURL

curl -G "https://france-energie.koyeb.app/api/v1/consommation_chauffage" \
    --data-urlencode "surface_habitable_logement=75" \
    --data-urlencode "altitude=125" \
    --data-urlencode "annee_construction=2010" \
    --data-urlencode "zone_climatique=H1a" \
    --data-urlencode "type_batiment=appartement" \
    --data-urlencode "qualite_isolation_enveloppe=très bonne" \
    --data-urlencode "type_energie_principale_chauffage=Bois – Bûches" \
    --data-urlencode "logement_traversant=1" \
    --data-urlencode "protection_solaire_exterieure=1"

Python

import requests

base_url = "https://france-energie.koyeb.app/api/v1"
endpoint = "/consommation_chauffage"

params = {
    'surface_habitable_logement': 75,
    'altitude': 125,
    'annee_construction': 2010,
    'zone_climatique': 'H1a',
    'type_batiment': 'appartement',
    'qualite_isolation_enveloppe': 'très bonne',
    'type_energie_principale_chauffage': 'Bois – Bûches',
    'logement_traversant': 1,
    'protection_solaire_exterieure': 1
}

response = requests.get(base_url + endpoint, params=params)
